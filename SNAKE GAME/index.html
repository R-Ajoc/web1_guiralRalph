SNAKE GAME

HOW TO PLAY?

The Snake Game is a classic arcade-style game where the player controls a snake that moves around the screen, eating food and growing longer while avoiding collisions with itself and the walls. Here’s how to play:

  1. Start the game – The snake starts as a small line moving in a specific direction.
  2. Move the snake – Use arrow keys (⬆️⬇️⬅️➡️) keys to control its movement.
  3. Eat food – When the snake eats a food item, it grows longer.
  4. Avoid crashing – The game ends if:
    • The snake collides with the walls.
    • The snake collides with its own body.
    • The snake collides with the obstacles (Extreme Mode).
